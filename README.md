# unity-scripts
## ClickMe  
Click Events (Mouse Events), equivalent to MouseDown, MouseUp etc but for Input System Package  

## EventManager & ClickMeReaction  
Mini Event-Manager for sending Events to other Gameobjects  

## EndWithEsc  
A Script that ends Unity Playmode in the Editor or in the Build with pressing Esc  

## PlayerMove  
Basic 2D movement with transform.position, no physics  

## RestartScene  
A Script that restarts the current Scene by pressing R  

## SoundManager  
- Basic Sound Manager, OnAwake loading Soundfiles of “Assets/Ressources/Sounds” to a list  
- Play Soundfile with its name (String) via any other Script
